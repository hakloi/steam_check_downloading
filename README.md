# Quick Start:

1) Clone repository, write this code in your terminal:
`git clone https://github.com/hakloi/steam_check_downloading.git`
2) Change directory:
`cd steam_check_downloading`
3) Run script: `python script.py`
4) The script offers 3 modes: once, five, and continuous monitoring.* Write the needed mode in your terminal.

- Once - single check;
- Five - 5-minutes monitoring;
- Continuous - continuous monitoring.

Be aware that you need Steam account and active internet connection!
